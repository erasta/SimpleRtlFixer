# Simple RTL Fixer

Firefox extension that fixes RTL text direction on any web page with a single click.

## How it works

Click the toolbar button to apply `dir="auto"` to text elements on the current page. New content that loads afterwards is fixed automatically.

## Install

- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/simple-rtl-fixer/)

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

Creates a `.zip` in `web-ext-artifacts/` ready for submission.
